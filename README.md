# WhetherWeather
* Full Stack App determining what the weather is like in a safe and extremely lightweight way, from an outside API.

    Front-End: 
         Framework - React
         Data structure - Redux
         API fetch- Axios
    Back-End: 
         Framework Base - Node
         Server API -  Express
         API fetch- Axios
